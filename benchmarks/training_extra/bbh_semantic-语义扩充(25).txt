    result = []
    i = j = 0
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            result.append(a[i]); i += 1

Q106: Write a Python function to implement Dijkstra's shortest path algorithm.
答案: import heapq
def dijkstra(g, s):
    dist = {s: 0}
    pq = [(0, s)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist.get(u, float('inf')): continue
        for v, w in g.get(u, {}).items():
            if d + w < dist.get(v, float('inf')):
                dist[v] = d + w
                heapq.heappush(pq, (d + w, v))
    return dist

Q122: Explain the concept of the Composite design pattern.
答案: The Composite pattern composes objects into tree structures to represent part-whole hierarchies, treating individual objects and compositions uniformly. Example: a file system.

Q123: What is the difference between a mutex and a semaphore?
答案: A mutex allows only one thread to access a resource at a time (binary). A semaphore allows N threads to access a resource (counting), and can be used for resource pooling.

Q124: Write a Python function to implement a bloom filter.
答案: import mmh3
class BloomFilter:
    def __init__(self, size=1000, k=3):
        self.bit = [0] * size
        self.k = k
    def add(self, x):
        for i in range(self.k):
            self.bit[mmh3.hash(x, i) % len(self.bit)] = 1

Q125: Explain the concept of the Builder design pattern.
答案: The Builder pattern separates the construction of a complex object from its representation, building it step by step. The same construction process can produce different representations, e.g., an SQL query builder.
